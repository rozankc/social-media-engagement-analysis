# Social Media Engagement Analysis

## Project Overview
This project analyzes social media engagement patterns using data visualization and analytical techniques. The goal is to identify how factors such as sentiment, emotion, time, platform, and region influence user engagement.

## Objectives
- Identify key drivers of engagement across social media platforms
- Analyze how emotional tone impacts engagement rates
- Determine optimal posting times and campaign strategies
- Compare engagement across platforms and regions

## Dataset
- Source: Kaggle (Social Media Engagement Dataset)
- Features:
  - Likes, Comments, Shares, Impressions
  - Sentiment & Emotion
  - Campaign Phase
  - Platform (Facebook, Instagram, Twitter, Reddit)
  - Timestamp & Location

## Tools & Technologies
- Tableau (Dashboard & Storytelling)
- Excel (Data Cleaning & Preparation)
- SQL (Data Structuring - optional if used)
- Data Visualization Techniques

## Key Analysis Performed
- Engagement Rate Calculation:
  (Likes + Comments + Shares) / Impressions
- Time-based analysis (hour, weekday, campaign phase)
- Sentiment & emotion impact analysis
- Platform comparison
- Geographic engagement analysis

## Key Insights
- Positive emotions (Happy, Excited) drive higher engagement
- Evening posts (6–9 PM) perform best on Facebook & Instagram
- Engagement peaks during launch and post-launch phases
- Sentiment varies significantly across platforms (Twitter & Reddit more volatile)
- Countries like South Korea and Canada show higher engagement rates

## Dashboard Features
- Interactive Tableau dashboards
- Heatmaps, time-series charts, and boxplots
- Viz-in-Tooltip for deeper insights
- Storytelling dashboards for decision-making
